#### SFTP

可以免费试用，但是会有弹窗出来提示你购买，所以在网上找了一个万能 *Key*，放在 `SFTP -> Settings - User` 里就好。

```json
{
  "email": "xiaosong@xiaosong.me",
  "product_key": "d419f6-de89e9-0aae59-2acea1-07f92a"
}
```

现在我已经没有弹窗提示了。
